# DVA-oriented resume

This folder collects resume links and notes that highlight the capstone work (EDA, forecasting, and Tableau dashboards).

## What is here

- Individual markdown files with each person's resume link.
- A consolidated index in `resumes.json` for quick, readable access.

## How to update

1. Add or edit a person's `.md` file with the latest link.
2. Keep `resumes.json` in sync so dashboards or scripts can consume it.

## Note

This folder is not used by the Python application.
